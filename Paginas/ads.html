O Curso de ADS

O curso Superior de Tecnologia em Análise e Desenvolvimento de Sistemas do IFCE Campus Boa Viagem forma profissionais capazes de analisar, projetar, desenvolver, testar e manter sistemas computacionais para diferentes áreas de atuação.

A formação combina conhecimentos teóricos e práticos, preparando os estudantes para atuar no mercado de tecnologia por meio do desenvolvimento de soluções inovadoras e alinhadas às demandas da transformação digital.

Eixos Disciplinares

01

Desenvolvimento de Software

Criação de aplicações web, desktop e mobile utilizando linguagens, frameworks e ferramentas modernas.

02

Banco de Dados

Modelagem, implementação e gerenciamento de bancos de dados para armazenamento e processamento de informações.

03

Engenharia de Software

Aplicação de metodologias, processos e boas práticas para o desenvolvimento de sistemas de qualidade.

Disciplinas do Curso

Conheça o objetivo e as competências desenvolvidas em algumas das principais disciplinas da matriz curricular do curso. Clique em cada uma para ver os detalhes.

Objetivo: desenvolver a capacidade de modelar e implementar sistemas usando os conceitos de classes, herança, polimorfismo e encapsulamento.
Competências: lógica de programação avançada, organização de código, reuso e manutenção de software.
Área relacionada: Desenvolvimento de Software.

Objetivo: capacitar o estudante a modelar, projetar e implementar bancos de dados relacionais eficientes e seguros.
Competências: modelagem entidade-relacionamento, normalização, SQL e administração de SGBDs.
Área relacionada: Banco de Dados.

Objetivo: apresentar processos, metodologias ágeis e boas práticas para o planejamento e a gestão do ciclo de vida de sistemas de software.
Competências: levantamento de requisitos, gestão de projetos, trabalho em equipe e qualidade de software.
Área relacionada: Engenharia de Software.

Objetivo: formar profissionais aptos a criar aplicações web completas, do front-end ao back-end.
Competências: HTML, CSS, JavaScript, frameworks modernos, APIs e integração com bancos de dados.
Área relacionada: Desenvolvimento de Software.

Objetivo: compreender os fundamentos de comunicação de dados, protocolos e arquitetura de redes.
Competências: configuração de redes, segurança básica da informação e infraestrutura de TI.
Área relacionada: Infraestrutura e Redes.

Objetivo: estudar formas eficientes de organizar e manipular dados na memória do computador.
Competências: listas, pilhas, filas, árvores, grafos e análise de complexidade de algoritmos.
Área relacionada: Fundamentos da Computação.

Objetivo: desenvolver habilidades de planejamento, execução e controle de projetos tecnológicos.
Competências: metodologias ágeis (Scrum, Kanban), gestão de equipes, prazos e riscos.
Área relacionada: Engenharia de Software.

Objetivo: introduzir conceitos de aprendizado de máquina e suas aplicações práticas no desenvolvimento de sistemas inteligentes.
Competências: análise de dados, algoritmos de classificação e introdução a bibliotecas de IA.
Área relacionada: Desenvolvimento de Software.
