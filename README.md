# Portal IFCE Campus Boa Viagem

Projeto acadêmico desenvolvido para a disciplina de Tecnologia Web.

## Sobre o Projeto

O portal tem como objetivo apresentar informações sobre o IFCE Campus Boa Viagem, incluindo cursos, formas de ingresso, pesquisa, extensão, bolsas e notícias.

## Tecnologias Utilizadas

- HTML5
- CSS3

## Estrutura do Projeto

- Página Inicial
- Conheça o IFCE
- Campus Boa Viagem
- Cursos
- Ingresso
- Pesquisa
- Extensão
- Bolsas
- Notícias
- Contato
