# Portal IFCE Campus Boa Viagem

Projeto acadêmico desenvolvido para a disciplina de Tecnologia Web, com o objetivo de apresentar informações institucionais do Instituto Federal de Educação, Ciência e Tecnologia do Ceará (IFCE) - Campus Boa Viagem.

## Sobre o Projeto

O Portal IFCE Campus Boa Viagem foi desenvolvido para centralizar e disponibilizar informações relevantes sobre a instituição, seus cursos, formas de ingresso, projetos acadêmicos e oportunidades oferecidas aos estudantes.

O site busca proporcionar uma navegação intuitiva, moderna e responsiva, permitindo que visitantes conheçam melhor a estrutura e os serviços oferecidos pelo campus.

## Funcionalidades

* Apresentação institucional do campus
* Informações sobre cursos ofertados
* Formas de ingresso
* Projetos de pesquisa e extensão
* Programas de bolsas e assistência estudantil
* Notícias e informações acadêmicas
* Página de contato

## Tecnologias Utilizadas

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Design Responsivo

Este projeto foi desenvolvido como atividade prática da disciplina de Tecnologia Web, aplicando conceitos de estruturação semântica com HTML, estilização com CSS, responsividade e organização de interfaces para web.
